# AirWatch 

Live air quality and traffic correlation dashboard for the Kondapur–Gachibowli 
IT corridor, Hyderabad.

**Live dashboard:** https://geekified05.github.io/airwatch/

## About
This is the application output of a B.Tech Real-time Research Project titled  
*"Time-Based Correlation Analysis of Traffic Congestion and Indoor PM2.5/PM10 
Air Pollution Levels"* conducted at Mahatma Gandhi Institute of Technology, 
Hyderabad (2025–26).

The dashboard reads live data from an automated hourly data logger that 
simultaneously collects traffic congestion data (Google Maps Directions API) 
and air quality readings from 5 sources: OpenWeatherMap, IQAir, AQINow, 
AQI.in, and Plume Labs.
